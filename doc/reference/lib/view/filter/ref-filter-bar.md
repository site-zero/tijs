# `TiFilterBar` 过滤条

## 控件概述

`TiFilterBar` 提供一个过滤条件的展示界面。它分作下面几个展示部分:

1. `major`: 主条件
2. `more`: 其余条件
3. 弹出的全部控件编辑器

它有下面几种布局形态

```bash
#
# 强制一行: NOWRAP
#
+----------------+----------------------+
| {xxxx} {xxxxx} | [Tag1] [Tag2] [Tag3] |
+----------------+----------------------+
    # Major              # More
#
# 折行显示: WRAP-MORE
#
+--------------------------+   #
| {xxxx} {xxxxx}           |   # Major
+--------------------------+   #
| [Tag1] [Tag2] [Tag3] |   |   # More
+--------------------------+   #
```
